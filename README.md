# Somay Khatri — Developer Portfolio Website

A personal developer portfolio website designed with inspiration from [heynesh.com](https://heynesh.com/), showcasing cloud engineering, DevOps automation, and full-stack MERN projects.

## 🚀 Features

- **Morphing Sticky Navigation**: Transforms from floating hero state to a compact dark pill header with logo badge, active page tracking, marquee, and statistics counters as you scroll down.
- **Group Hover Card Dimming**: Work cards dim with an overlay when hovering over any project card.
- **Character Reveal Heading**: GSAP character-by-character text reveal animation.
- **Interactive Counters**: Animated count-up statistics for CGPA, pipelines, and internships.
- **FAQ Accordion**: Animated expanding answer drawers with rotating icons.
- **Clipboard Email Copy**: Click-to-copy email address with instant feedback.
- **Responsive Layout**: Designed for mobile, tablet, and desktop viewports.

## 🛠️ Built With

- **HTML5 & Vanilla CSS**: Custom tokens, grid/flexbox layouts, responsive design
- **JavaScript (ES6+)**: Custom DOM interactions
- **GSAP & ScrollTrigger**: Scroll-driven timelines and text reveal animations
- **Lenis**: Ultra-smooth scrolling engine
- **Google Fonts**: Outfit & Space Grotesk typography
